The Amethyst Game Engine

Unleach your imagination with the most powerful community made, data-driven game engine

High Performance

Being built entirely on the powerful entity component system (ECS) architecture, Amethyst has performance that will last you through any project no matter what your plans are. The systems automatically utilize all processing power available, making sure your game stays performant.

Built for 2D and 3D

No matter if you want to make a 2D or a 3D game, Amethyst has you covered with many utilities and a fast and modular rendering pipeline that works seamlessly with both dimmensions.

A Stable Foundation

Because Amethyst is made using the Rust programming language it can almost garantee that your games will never crash or encounter unpleasant bugs.
